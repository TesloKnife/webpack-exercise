# 🎵 Weather Sound

Приложение для релаксации с звуками природы. Выберите погоду (лето, дождь, зима) и наслаждайтесь атмосферной музыкой с красивым визуальным оформлением.

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Команды

| Команда             | Описание                                     |
| ------------------- | -------------------------------------------- |
| `npm start`         | Запуск dev-сервера (http://localhost:3001)   |
| `npm run build:dev` | Сборка для разработки (с source maps)        |
| `npm run build`     | Production сборка (минификация, оптимизация) |
| `npm run lint`      | Проверка кода ESLint                         |
| `npm run lint:fix`  | Автоматическое исправление ошибок ESLint     |

## 🎮 Как использовать

1. Нажмите на одну из карточек погоды
2. Начнёт играть соответствующая музыка
3. Фон страницы размоется и станет соответствовать выбранной погоде
4. Используйте ползунок для регулировки громкости
5. Повторное нажатие ставит на паузу, ещё раз — возобновляет
6. Переключение между карточками останавливает предыдущий трек

## 🛠 Технологии

- **Webpack 5** — сборка проекта
- **SCSS** — стилизация
- **Vanilla JS** — логика приложения
- **ESLint** — линтинг кода
